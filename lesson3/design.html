Typography

Heading Font: Georgia

I chose the Georgia font, because it is a Web safe font. And I wanted a serif font as heading font.

Paragraph Font: Helvetica

I chose the Helvetica font, because it is a Web safe font. And I wanted a sans-serif font as paragraph font to match with the Georgia font.
